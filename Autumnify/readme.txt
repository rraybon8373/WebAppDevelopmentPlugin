== Description ==
This plugin will change the color and background color of every element on the web page to a color correlated to autumn. this is achieved by calculating the brightness of each color and changing the color to a color in the gradient with the closest brightness. this plugin looks best on a site with multiple colors.
== Installation == 
1. Upload the folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. The plugin will then be installed.
